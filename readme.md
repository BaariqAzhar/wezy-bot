# WA Bot

Learn how to use Node JS, using whatsapp-web.js to connect to WA

## Tech Stack

**Backend** NodeJS, whatsapp-web.js, qrcode-terminal, etc

## Run Locally

Clone the project

```bash
  git clone (url)
```

Go to the project directory

```bash
  cd (folder name)
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  node app.js
```
